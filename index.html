ThunderUp

A fork of StayPuft, which is a fork of Casper, the default theme for Ghost.

Features

  • Responsive design.
  • Post comments using Disqus.
  • In-site search using GhostHunter.
  • Support for Font Awesome.
  • Basic support for slidr.js.
  • Syntax highlighting using Prism.

Demo

This theme is being used in my blog.
